# Expense Tracker System

This is a Python-based Expense Tracker that helps users record and manage their daily expenses.
The project was built to practice Python concepts such as file handling, modular programming, exception handling, and data processing.

The application stores expense records in a CSV file and provides different reporting features to analyze spending patterns.

# Features

- Add new expenses
- View all expenses
- Search expenses by category
- Update existing records
- Delete records
- Budget management
- Remaining budget tracking
- Dashboard summary

# Technologies Used

- Python
- CSV File Handling
- Exception Handling
- Modular Programming

# Project Structure

```text
main.py        -> Main application
expense.py     -> Expense operations
reports.py     -> Reports and dashboard
budget.py      -> Budget management
config.py      -> Configuration variables
expenses.csv   -> Expense records
budget.txt     -> Stores budget value
```

# Why I Built This Project

I wanted to build a practical Python project instead of only solving coding problems. 
Through this project, I learned how to organize code into modules, work with CSV files, implement CRUD operations, and generate reports from stored data.
